Create `llm_benchmark/menu.py` with:

1. `run_interactive_menu(models: list) -> argparse.Namespace` function:
   - Use `get_console()` for all output (never `print()`)
   - Show system info one-liner via `format_system_summary()` (lazy import from system.py)
   - Print numbered menu:
     ```
     LLM Benchmark

       1. Quick test (~30 seconds)
       2. Standard benchmark
       3. Full benchmark
       4. Custom
     ```
   - Loop on `input("Select mode [1-4]: ")` until valid; on invalid print "  Please enter 1-4"
   - Wrap `input()` in try/except for EOFError and KeyboardInterrupt (exit cleanly with sys.exit(0))
   - Mode logic:
     - Quick test: sort models by `m.size` ascending, pick first. prompt_set="small" (but override prompts with single short prompt: "Write a one-sentence summary of what a CPU does."), runs_per_prompt=1, skip_warmup=True. Set skip_models to exclude all but the smallest.
     - Standard: prompt_set="medium", runs_per_prompt=2, skip_warmup=False
     - Full: prompt_set="large", runs_per_prompt=3, skip_warmup=False
     - Custom: prompt user for prompt set (1=small, 2=medium, 3=large with input loop), runs_per_prompt (input with default 2), model selection (numbered list, "Skip models (e.g. 3,4) or Enter for all:")
   - Return `argparse.Namespace` with all fields _handle_run expects: verbose=False, skip_checks=True (preflight already ran), skip_models, prompt_set, prompts (None except quick test), runs_per_prompt, timeout=300, skip_warmup, max_retries=3, concurrent=None, sweep=False, command="run", debug=False

2. Create `llm_benchmark/display.py` with:

   a. Constants: `BAR_FULL = "\u2588"`, `BAR_EMPTY = "\u2591"`, `BAR_WIDTH = 30`

   b. `render_bar_chart(rankings: list[tuple[str, float]], metric_label: str = "t/s") -> None`:
      - Use `get_console()` for output
      - If empty rankings, return
      - Find max_rate and max_name_len
      - Print header "[bold]Model Rankings[/bold]"
      - For each (name, rate): compute bar_len proportional to max_rate, print formatted line with full/empty blocks and rate
      - Print recommendation: "Best for your setup: [bold]{name}[/bold] ({rate:.1f} {metric_label}) -- fastest response generation"

   c. `render_text_bar_chart(rankings: list[tuple[str, float]], metric_label: str = "t/s") -> str`:
      - Same logic as render_bar_chart but returns plain string (no Rich markup)
      - For Markdown reports: includes model name, bar, rate on each line
      - Includes recommendation line at end
      - No ANSI codes -- pure Unicode text
