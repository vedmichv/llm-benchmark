    3. `chat(model, messages, stream=False, options=None)`:
       - Non-streaming: calls `ollama.chat()`, converts response via `_to_response()`, returns BackendResponse
       - Streaming: calls `ollama.chat(stream=True)`, returns StreamResult where chunks yields content strings and finalize builds BackendResponse from final chunk
       - Wraps all ollama exceptions into BackendError with appropriate retryable flag:
         - ConnectionError -> retryable=True
         - ollama.RequestError -> retryable=True
         - ollama.ResponseError with status >= 500 -> retryable=True
         - ollama.ResponseError with status < 500 -> retryable=False
       - options dict passed directly to ollama.chat (ollama SDK accepts dicts)
    4. `list_models()`: calls ollama.list(), returns list of dicts with at minimum 'model' and 'size' keys
    5. `unload_model(model)`: calls ollama.generate(model=model, prompt="", keep_alive=0), returns True/False
    6. `warmup(model, timeout)`: calls ollama.chat with short prompt, returns True/False (no console output -- runner handles display)
    7. `detect_context_window(model)`: calls ollama.show(), extracts context_length from modelinfo, returns int (default 4096)
    8. `get_model_size(model)`: calls ollama.list(), finds model, returns size in GB (float), or None
    9. `check_connectivity()`: calls ollama.list(), returns True/False
    10. Private `_to_response(raw_response)`: converts ollama response dict (nanoseconds) to BackendResponse (seconds). Divides all duration fields by 1_000_000_000. Detects prompt_eval_count == -1 as cached, normalizes to 0.
